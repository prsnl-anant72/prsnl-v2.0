# Kittu Birthday App

A romantic photo gallery and countdown app created for Kittu, built with React, Tailwind CSS, and Framer Motion.

## Features

- Beautiful photo gallery with romantic images
- Animated countdown timer
- Responsive design with smooth animations
- Built with modern React and Tailwind CSS

## Technologies Used

- React 18
- Tailwind CSS
- Framer Motion for animations
- Lucide React for icons

## Local Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Deployment

This app is configured for deployment on Render. The build process and deployment configuration are set up in `render.yaml`.

To deploy:
1. Connect your GitHub repository to Render
2. Render will automatically detect the `render.yaml` configuration
3. Deploy the static site

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.
